## DuckDB SQL tips
- Use CAST(x AS DOUBLE) or x * 1.0 for float division.
- NULLs: aggregate functions ignore NULLs.
- String comparison is case-sensitive. Use ILIKE for case-insensitive.
- Dates: DATE_PART, EXTRACT, DATE_TRUNC, DATEDIFF.
